# Number-Guessing-Game

A simple Python number guessing game where the computer randomly selects a number, and the player has a limited number of attempts to guess it correctly.

## ✨ Features

- Three difficulty levels::
  - Easy
  - Medium
  - Hard
- Random number generation
- Limited number of attempts
- Input validation
- Clean command-line interface

## 📁 Project Structure

```text
.

```

## 🚀 Installation

Prerequisites

```bash
Python 3.10 or newer
```

Clone the repository:

```bash
git clone https://github.com/danielmascian/Number-Guessing-Game.git
```

Move into the project directory:

```bash
cd Number-Guessing-Game
```

Run the Game:

```bash
python main.py
```

## 🎮 Gameplay

- Choose a difficulty level.
- The computer generates a random number.
- Enter your guesses.
- Receive hints after each attempt.
- Win by guessing the correct number before running out of attempts.
